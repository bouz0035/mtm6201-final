# mtm6201-final

## Project Overview
This project is a multi-page responsive restaurant website created with Bootstrap. The design was based on my Figma mockups and adapted into a working website with reusable styling and consistent layout across pages.

## Pages Included
- Home
- Menu
- Reservation

## Process
I started by reviewing my Figma layouts and choosing the strongest three pages to build for the final website. I focused on keeping the design consistent by using one shared CSS file and repeating the same navigation, footer, colours, and spacing system across the pages.

I used Bootstrap for layout structure, spacing, and responsive behaviour. I also customized the site colours using CSS variables so the site felt closer to my visual design.

## Challenges and Solutions
One challenge during development was keeping the layout clean across different screen sizes. I solved this by testing the website at mobile, tablet, and desktop widths and adjusting spacing, grid layout, and image scaling.

Another challenge was making the pages feel visually consistent while still having different content. I solved this by building reusable card styles, buttons, and section patterns.

I also had to think carefully about accessibility, especially semantic structure, alt text, form labels, and keyboard navigation.

## What I Learned
Through this project I learned how to turn a design idea into a functional responsive website. I improved my understanding of Bootstrap layouts, CSS variables, reusable styling, responsive images, and accessible page structure.
